## 相关css属性

* IE浏览器, 火狐浏览器  
  * `scrollbar-*`系列的属性
  > 里边有的css属性支持火狐浏览器，有的支持IE
  
* 谷歌浏览器，safari浏览器，edge浏览器
  * `::-webkit-scrollbar`  
     > 可设置滚动条的宽度
  * `::-webkit-scrollbar-track-piece` 
    > 可设置滚动条轨道的颜色，边角样式
  * `::-webkit-scrollbar-thumb` 
    > 可设置滚动块的颜色，边角样式
